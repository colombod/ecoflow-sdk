# ecoflow-sdk

SDK for controlling and monitoring EcoFlow energy devices.

## Libraries

| Language | Package | Status |
|----------|---------|--------|
| **Python** | [`ecoflow-python`](./python/) | ✅ Available |
| TypeScript | `ecoflow-ts` | 🔜 Planned |

## Python Quick Start

```bash
pip install ecoflow-python
```
